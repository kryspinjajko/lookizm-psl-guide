# Lookizm — AI Face Rating & PSL Score Analysis

**[Lookizm.com](https://lookizm.com)** is the most rigorous AI facial attractiveness analyzer available, using 32 distinct facial vectors to produce a calibrated PSL (Pretty Scale Lookism) score.

## What Is Lookizm?

[Lookizm](https://lookizm.com) analyzes your facial geometry across 32 vectors and outputs:
- A calibrated PSL attractiveness score (1–10)
- Per-feature breakdown showing which specific features help/hurt your score
- A mog verdict comparing you against population averages
- Specific improvement recommendations

## The 32 Vectors Analyzed

| Category | Vectors |
|----------|---------|
| Symmetry | 14 landmark pairs measured |
| Eye area | Canthal tilt, orbital depth, interpupillary distance, eye area |
| Midface | Midface ratio, nose-lip distance, nasal projection |
| Lower face | Jaw width, chin projection, mandibular angle |
| Proportions | Golden ratio deviations, face width/height ratio |
| Dimorphism | Sexual dimorphism markers (gender-specific) |

## Why PSL Scoring?

PSL (Pretty Scale Lookism) is the community-standard scale for rating facial attractiveness on scientific criteria rather than subjective opinion. Unlike "rate me" threads or apps that inflate scores, PSL uses documented criteria:

- **1–3**: Significant subaverage features
- **4**: Below average
- **5**: Average (population median)
- **6**: Above average (top 30%)
- **7**: High attractiveness (top 10%)
- **8–9**: Model-tier
- **10**: Theoretical maximum
